The 5-step pipeline, executed¶

Step What I did
ABSORB Read the Compléte Manual Part 6 (return of Phát Bóng) + the Phát Bóng deep dive (the Phát Bóng is the return's mirror) + the 2 skill references (bundled-vs-single-stroke, OHB models — neither applied directly but informed the structure)
EXTRACT Pulled 25+ concepts across reading, block, cú đẩy, and 50+ concerns → mapped to 11 cues, 7 errors, 5 bài tậps, 6 tai chi principles

SYNTHESIZE Master huấn luyện viên Giọng nói, taichi parallels (Lã Kình on block = yield & redirect, Phóng Tùng on cú đẩy = release tension), 50+ cues (shoulder, brás-hand elbow, eyes 2-3 ms slower, bridge leg)
FORMAT EN-VI side-by-side tables, decision tree (when to block vs cú đẩy), 50+ reality check, printable 1-page cheat sheet (EN + VI)
### Key 3.5-level choices I made
- Made reading Chapter 2, not Chapter 1 — because Ý Tưởng Chính is the survival mindset (Ch 1), but the technical foundation is reading. Most deep dives put reading late; I put it early so the user can thực hành it from day 1.
- Wove the cú nhẹ/cắt return into the block chapter as a "variation" — because at 50+, the cú nhẹ is essentially a block with a small xoáy dưới component. Splitting it into its own chapter would have over-emphasized it.
- Used the "70% swing" rule on the cú đẩy return — the second Phát Bóng doesn't need a full L-release swing. A shorter, controlled swing is the 3.5 reality.
- Translated the "physics of the toss" into body cues ("front = flat, side = cắt, behind = kick") instead of frame counts or joint angles. The user is 50+; the eyes need the simple map.
- Added the "20-bóng block" service-line bài tập — this trains the block return WITHOUT needing actual phát bóngs. It can be done at any club with a hitting partner.
- Ended with a printable 1-page cheat sheet in both languages, with the "bịch vs cốc" sound cue included (Surrey bag-ready).
